>
> **Goal:** 在项目中初始化两组核心词库：语文宝库（50 个一年级高频生字，按关卡组织）与数学探险（覆盖一年级常见题干关键词/短语，解决读题理解痛点），并把第 1 关数据从硬编码切换为读取词库。

>
> **Tech Stack:** Taro 4 + React + TypeScript + TailwindCSS(weapp-tailwindcss) + Vitest

```ts
export function toBlankSentence(sentence: string, target: string) {
  if (!target) return sentence
  const idx = sentence.indexOf(target)
  if (idx < 0) return sentence
  return sentence.slice(0, idx) + '___' + sentence.slice(idx + target.length)
}
```
